# Jeu de plateau Othello en Java
Ceci est un projet universitaire réalisé en début de 1ère année de BUT Informatique.  
Il s'agit d'une implémentation du jeu de plateau Othello.

## Contributeurs
- Anas SEMGHOUNI
- Wael BEN MOUSSA

## Remarques
- Ce projet a été réalisé avant l’introduction de la programmation orientée objet.
  Il reflète un stade d’apprentissage antérieur.
- Le développement ayant été réalisé en binôme, la collaboration s’est faite localement ;
  l’utilisation de Git n’était pas encore maîtrisée à ce stade.

## Fonctionnalités
- Plateau de jeu Othello
- Gestion des règles de base
- Mode deux joueurs
- Mode contre une IA simple
- Comptage des pions

## Exécution
Le projet peut être exécuté après clonage du dépôt, en lançant la classe principale depuis un IDE Java.
