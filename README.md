# Jeu de plateau Othello en Java
Ce projet est un **travail universitaire** réalisé en début de **1ère année de BUT Informatique**.  
Il s'agit d'une implémentation du jeu de plateau **Othello**, développée en Java.

## Contributeurs

- Anas SEMGHOUNI
- Wael BEN MOUSSA

## Contexte et Remarques

- Ce projet a été réalisé **avant l’introduction de la programmation orientée objet**.  
  Il repose donc sur une approche **procédurale**, représentative d’un stade
  d’apprentissage initial.
- Le développement ayant été réalisé en binôme, la collaboration s’est faite localement ;
  l’utilisation de **Git** n’était pas encore maîtrisée à ce stade.

Ce projet illustre ainsi les premières bases de la programmation en Java, notamment la
logique algorithmique et la gestion des règles d’un jeu.

## Fonctionnalités

- Plateau de jeu Othello
- Gestion des règles de base
- Mode deux joueurs
- Mode contre une IA simple
- Comptage des pions

## Technologies utilisées

- Java
- Programmation procédurale
- IDE Java (IntelliJ IDEA)

## Exécution du projet

Le projet peut être exécuté de la manière suivante :

1. Cloner ou télécharger le dépôt
2. Ouvrir le projet dans un **IDE compatible Java**
3. Repérer la classe contenant la méthode `main`
4. Lancer l’application depuis l’IDE

Aucune bibliothèque externe, base de données ou outil supplémentaire n’est requis.

## Note pédagogique

Ce projet représente une **étape d’apprentissage** et précède l’utilisation :
- de la programmation orientée objet,
- des architectures logicielles,
- des outils de build comme Maven.

Il permet néanmoins de poser les bases de la logique algorithmique et de la structuration d’un programme.
